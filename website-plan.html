Advanced Eye Care

Site Name

Advanced Eye Care Optometry

This name was chosen because it reflects the core purpose of the website: promoting better vision and eye health. It is simple, memorable, and relevant to the optometry field.

Site Purpose

The website serves as a hub for patients and visitors to learn about eye health, explore optometry services, and connect with the clinic. It provides:

  • Basic eye health tips and information.
  • A list of optometry services offered.
  • Interactive vision tests (e.g., color blindness test).
  • An appointment request form and clinic location via Google Maps.

Scenarios

Target audience questions:

  1. What services does the clinic offer for routine eye exams?
  2. How can I request an appointment online?

Color Schema

The following colors will be used:

  • Primary Color (#0077B6): Used for headings, navigation, and accents.
  • Secondary Color (#F4A261): Used for buttons, highlights, and interactive elements.
  • Background Color (#F8F9FA): Used for the website background.
  • Text Color (#333333): Used for body text.

Typography

The following fonts will be used:

  • Merriweather: Used for headings (h1, h2, h3).
  • Roboto: Used for body text and general content.

Wireframe

Mobile View:

  • Single-column layout with stacked sections.
  • Header at the top, followed by the main content sections, and footer at the bottom.

Desktop View:

  • Two-column layout for the main content, with the header and footer spanning the full width.
  • Sections like "Site Name" and "Site Purpose" will be side by side.
